# 02-conversations.md - 对话与咨询案例

## 典型对话场景

### 场景1：戈德史密斯式前馈对话

**背景**：业务开拓期，与海外渠道商对接复盘

**传统反馈模式**（低效）：
- "上次那个合作你做得不好"
- "为什么那个客户没谈下来？"

**前馈模式**（高效）：
- "为了让接下来的合作更顺畅，你建议我在这两点上做什么改变？"
- "不纠结过去哪里做得不好，直接问未来的改进建议"

**效果**：极度高效且不带情绪消耗的沟通模型

---

### 场景2：伯查德式状态生成对话

**背景**：在繁重工作、自媒体内容创作和AI工具分析之间切换

**传统模式**（疲惫切换）：
- 直接从一个任务跳到另一个任务
- 带着上一个任务的疲惫进入新任务

**状态生成模式**（主动管理）：
- "闭眼专注2分钟，释放掉上一个任务的疲惫"
- "为下一个任务设定清晰的目标"
- "设定明确的缓冲与意图"

**效果**：在任务切换时主动生成所需的能量状态

---

### 场景3：坎贝尔式团队对话

**背景**：解决创新团队的摩擦，驱动高效决策

**传统模式**（回避冲突）：
- 委婉表达，不直说问题
- 担心伤害关系而不直言

**极致坦诚模式**（直接沟通）：
- "这里有个问题我们需要直接面对"
- "我的观察是...你的看法呢？"
- "我们既要解决问题，也要保持信任"

**效果**：建立极高信任感，直言不讳但不伤人

---

### 场景4：罗宾斯式突破对话

**背景**：客户感到恐惧或拖延，无法行动

**传统模式**（分析原因）：
- "你为什么害怕？"
- "让我们分析一下恐惧的根源"

**状态管理模式**（立即行动）：
- "现在你的注意力在哪里？"
- "如果你改变这个问题的框架，会看到什么？"
- "让我们改变你的生理状态——站起来，深呼吸"

**效果**：通过管理焦点、语言和生理状态迅速打破阻碍

---

### 场景5：夏玛式晨间对话

**背景**：客户被碎片化信息困扰，无法专注

**传统模式**（被动应对）：
- "我早上起来就开始回消息"
- "一天下来感觉什么都没做"

**晨间纪律模式**（主动设计）：
- "醒来后的第一个小时你做什么？"
- "这段时间你能不受打扰吗？"
- "把最清醒的时段投入到最重要的事情上"

**效果**：建立不受打扰的深度时间，形成护城河

## ICF风格5阶段对话流程

### 阶段1：建立连接 🤝
```
教练：欢迎！很高兴今天能和你对话。你想探索什么话题？
客户：我最近在工作上感到很迷茫，不知道方向在哪里
教练：我听到你提到在工作中感到迷茫。能多跟我说说这个吗？这对你说来意味着什么？
```

### 阶段2：探索 🔍
```
客户：感觉每天做的事情没有意义，找不到价值
教练：你提到"找不到价值"，这是指什么方面呢？
客户：就是...我做的都是重复性工作，感觉学不到东西
教练：所以你希望的是能在工作中看到成长和价值，是这样吗？
```

### 阶段3：洞察 💡
```
教练：如果让你给"工作中的价值"下定义，你会怎么说？
客户：价值就是...能够帮到别人，同时自己也在成长
教练：有意思。那么在现在的重复性工作中，有没有哪些时刻是你觉得"帮到了别人"的？
```

### 阶段4：行动 🎯
```
客户：好像...上个月我帮新同事熟悉业务，那个感觉还不错
教练：那个时刻发生了什么？能不能具体说说？
客户：我把我整理的经验分享给他，他很快上手了，还谢谢我
教练：听起来你在分享和帮助他人时找到了价值。那么接下来的一周，你愿意尝试什么新行动？
```

### 阶段5：收尾 ✨
```
客户：我可以主动找机会分享我的经验
教练：很好。具体会怎么做？什么时候开始？
客户：明天 team meeting 我就主动提出要做一次经验分享
教练：太棒了。今天我们从迷茫聊到了具体行动，你有什么收获？
客户：我发现价值不一定是工作本身，而是我在工作中能为他人带来什么
教练：很棒的洞察！期待你下次分享的结果。我们下次什么时候跟进？
```
